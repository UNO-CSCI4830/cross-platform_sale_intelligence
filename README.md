# Platform Sale Intelligence
By Group 4

To Run:
1. Clone the repository to your machine.
2. Create and activate a virtual environment(optional, but recommended).
    2.1.1: Windows: python -m venv venv
    2.1.2: Mac/Linux: python3 -m venv venv
        2.2.1. Windows: venv\Scripts\activate
        2.2.2. Mac/Linux: venv/bin/activate
3. Use "pip install requirements.txt" to install dependencies
4. Run the FastAPI server with "uvicorn backend.main:app --reload"
